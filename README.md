<h2 align="center">🌟  E-Banking Application  🌟</h2>

This project is a simple banking application developed with **Java**, **JDBC**, **Servlets**, **JSP**, **HTML**, and **CSS**. It provides essential banking features such as crediting, debiting, checking balances, transferring money, and changing passwords. The application follows a three-layer architecture for organized, modular code.

## ✨ Features

- 🔐 **User Registration**: Sign up to create a new account.
- 👤 **User Login**: Log in with your credentials.
- 💰 **Credit Amount**: Add funds to your account.
- 💸 **Debit Amount**: Withdraw funds from your account.
- 🔄 **Transfer Money**: Send funds to other accounts.
- 📊 **Check Balance**: View your account balance.
- 🔒 **Change Password**: Update your account password.

---

## 💻 Technologies Used

- **Backend**: Java, JDBC, Servlets
- **Frontend**: JSP, HTML, CSS
- **Database**: JDBC for database interactions
- **Session Management**: Session objects for user interactions across sessions

---

## 🏗️ Project Architecture

The project is organized into a **three-layer architecture**:

### Model Layer
- Contains **POJO (Plain Old Java Object) classes** that represent database tables.
- Facilitates data transfer between the application and the database.

### DAO Layer
- Manages **all database interactions** (CRUD operations).

### Servlet Layer
- Manages **HTTP requests and responses**.
- Implements core business logic for operations like credit, debit, transfer, and password changes.

---

## 📖 Usage

### User Registration
- Visit the registration page to create a new account.

### Login
- Log in using your registered credentials to access account operations.

### Banking Operations
- 💳 **Credit**: Add funds to your account.
- 🏧 **Debit**: Withdraw funds from your account.
- 🔀 **Transfer**: Send funds between accounts.
- 📈 **Check Balance**: View your current balance.
- 🔐 **Change Password**: Update your account password.

### Logout
- End the session to log out of your account.
